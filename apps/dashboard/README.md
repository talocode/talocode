# Talocode Dashboard

Vite + React + TypeScript + Tailwind CSS dashboard app for Talocode Cloud.

**Dashboard Preview** — placeholder UI showing what the full dashboard will look like. No real auth, API keys, or backend calls.

Deployed on Netlify at [dashboard.talocode.site](https://dashboard.talocode.site).

## Scripts

```bash
npm run dev      # Start dev server
npm run build    # Type-check and build for production
npm run preview  # Preview production build
```
