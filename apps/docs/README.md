# Talocode Docs

Vite + React + TypeScript + Tailwind documentation app for the Talocode ecosystem.

Deployed on Netlify at [docs.talocode.site](https://docs.talocode.site).

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Stack

- Vite 6
- React 19
- TypeScript 5
- Tailwind CSS 4
